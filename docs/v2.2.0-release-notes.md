# Rockpile v2.2.0 — 功能设计方案 & 实现总结

## 版本定位

**从"配置驱动"到"零配置体验"** — 去掉 4 步引导向导，启动即用。Dashboard 从碎片化卡片重构为 Timeline 时间轴，像 Notchi 一样提供全面的时间感知。

---

## 一、功能设计方案

### 1. 零配置启动 (P0)

**问题**: 用户首次启动需要完成 4 步引导（选语言→选角色→配O₂→装插件），体验繁琐。ScreenSelector 返回 nil 时黑屏。

**方案**: `AppDelegate.autoSetupIfNeeded()` 自动完成全部配置：
- 检测系统语言 → 设置 `appLanguage`
- 检测 `~/.claude/` → 设置 `role = .local`
- 调用 `AIProviderDetector` → 设置 O₂ 参数
- 调用 `HookInstaller` → 安装 bash hook
- 设置 `setupCompleted = true`

**Hook 系统升级**: 从遗留 JS 插件 (`~/.rockpile/plugins/`) 迁移到 bash hook (`~/.claude/hooks/rockpile-hook.sh`)，直接注册到 `~/.claude/settings.json`。

### 2. 极简水塘 (P1)

**问题**: SpongeBob 风格的沙地+海草+石头装饰在小尺寸水塘中显得拥挤。

**方案**: 去掉 OceanFloorView、SeaweedCluster、DecorationView，只保留：
- 极简深蓝渐变背景（3色，比之前更暗）
- 稀疏气泡
- 像素精灵（绝对定位，自动缩放）
- 水面波纹

**生物定位**: 从 `ZStack(alignment: .bottom) + offset` 改为 `.position()` 绝对定位：
- 小龙虾: `(65%, 30%)` — 右上
- 寄居蟹: `(35%, 65%)` — 左下
- `spriteScale = min(1.0, pondH / 150)` 自动缩放

### 3. Timeline Dashboard

**问题**: 旧 Dashboard 信息碎片化 — DualSourceInfoRow + 双O₂条 + TokenConsumptionCard + 活动日志 + 足迹，认知负担大。

**方案**: 用单一的 `ActivityTimelineView` 替代所有旧组件：

```
── 14:32 ──────────────
│ 🐚 ~/rockpile
│ 💬 "帮我改登录页面"
│ 🧠 12s → 🔨 Read 0.3s
│ 🔨 Bash: xcodebuild 45s
│ ✅ build succeeded
│ [2m15s  1.2K in · 3.4K out]
── 14:15 ──────────────
│ 🦞 ~/api-server
│ ...
```

- 时间锚点分隔 session
- 左侧色线标识生物（绿=🐚 寄居蟹，黄=🦞 小龙虾）
- 事件内联显示时长
- 底部有 session 统计行

### 4. 智能 O₂ 条

**问题**: 固定双条布局浪费空间（两个生物可能用同一个 API）。

**方案**: `SmartOxygenBarView` 运行时检测 provider：
- `localProvider == remoteProvider` → 合并为 1 条
- `localProvider != remoteProvider` → 显示 2 条，各带生物图标+标签+ETA

### 5. 权限处理 (P5)

**方案**: Hook 脚本收到 `PermissionRequest` 后：
1. 发送到 socket → `PermissionHandler.addRequest()`
2. Notch 自动展开 → 显示 `PermissionBannerView`（工具名 + 输入摘要 + Allow/Deny + 倒计时）
3. 用户点击 → 写 `/tmp/rockpile-permission-<id>.json`
4. Hook 脚本轮询该文件 → 返回决策给 Claude Code
5. 5 分钟超时自动 deny

### 6. 对话解析 (P4)

**方案**: `ConversationParser` actor 增量解析 `~/.claude/projects/<dir>/<sessionId>.jsonl`，提取用户消息、助手回复、工具调用摘要。`ConversationView` 以聊天气泡风格显示。

### 7. Aseprite 流程 (P6)

**方案**: `scripts/export-sprites.sh` 调用 Aseprite CLI 从 `.ase` 源文件导出 sprite sheet + JSON metadata。`SpriteMetadata.swift` 解析帧位置和时长，`SpriteSheetView` 支持 metadata 驱动渲染。

## 二、技术架构

### 新增文件 (12)

| 文件 | 角色 |
|------|------|
| `HookInstaller.swift` | bash hook 安装 + settings.json 注册 |
| `ClaudeKeychainReader.swift` | OAuth token 读取 |
| `ConversationParser.swift` | JSONL 增量解析 (actor) |
| `PermissionHandler.swift` | 权限请求队列 + 超时 |
| `SpriteMetadata.swift` | Aseprite JSON 解析 |
| `ActivityTimelineView.swift` | 时间轴事件流 |
| `ConversationView.swift` | 聊天气泡 |
| `ConversationMessage.swift` | 消息模型 |
| `PermissionBannerView.swift` | 权限横幅 UI |
| `SessionListView.swift` | 多 session 列表 |
| `SmartOxygenBarView.swift` | 智能 O₂ 条 |
| `scripts/export-sprites.sh` | Aseprite CLI |

### 修改文件 (14)

| 文件 | 改动 |
|------|------|
| `AppDelegate.swift` | autoSetupIfNeeded() 零配置 |
| `AppSettings.swift` | needsOnboarding 简化 |
| `DesignTokens.swift` | 字体全局 +1~2pt |
| `HookEvent.swift` | +cwd, toolUseId, toolInput |
| `StateMachine.swift` | 路由 PermissionRequest, 传 cwd |
| `UsageQueryService.swift` | OAuth usage 查询 |
| `ExpandedPanelView.swift` | Timeline 替代旧 Dashboard |
| `PondView.swift` | 极简化 + 绝对定位 |
| `GroundSpriteView.swift` | 去掉贴地硬编码 |
| `SpotlightInputView.swift` | 终端风格 > 提示符 |
| `SpriteSheetView.swift` | metadata 驱动支持 |
| `NotchContentView.swift` | 水塘 30% + hover + 权限 overlay |
| `NSScreen+Notch.swift` | force unwrap 修复 |
| `TokenTracker.swift` | rateLimited 自动恢复 |

### 事件流

```
Claude Code hook event
    ↓
~/.claude/hooks/rockpile-hook.sh (bash)
    ↓ Unix socket
SocketServer → StateMachine → SessionStore → UI
    ↓                ↓              ↓
    ↓         PermissionHandler  ConversationParser
    ↓                ↓
    ↓         /tmp/rockpile-permission-<id>.json
    ↓                ↓
    ↓         Hook script polls → returns decision
```

## 三、版本对比

| 维度 | v2.1.x | v2.2.0 |
|------|--------|--------|
| 启动 | 4 步引导 | 零配置 |
| Hook | JS 插件 (legacy) | bash hook (native) |
| 水塘 | SpongeBob 沙地 | 极简深蓝 |
| Dashboard | 碎片卡片 | Timeline 时间轴 |
| O₂ 条 | 固定双条 | 智能合并 |
| 权限 | 无 | Notch 内 approve/deny |
| 对话 | 无 | JSONL 解析 + 气泡 |
| 字体 | 8-14pt | 9-15pt |
| 精灵定位 | offset | position() |
